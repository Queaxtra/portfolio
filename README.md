# Personal Portfolio Website

This is a modern, responsive portfolio website built with Svelte and TypeScript. It showcases my provides contact information, and offers a customizable user experience with theme.

## Features

- **Responsive Design**: Adapts seamlessly to different screen sizes
- **Dark/Light Mode**: Toggle between dark and light themes
- **Contact Information**: Easy access to various contact methods
- **Social Media Links**: Quick links to professional social media profiles

## Pages

1. **Home**: Introduces the developer with a brief description
2. **Contact**: Provides multiple ways to get in touch

## Components

- **Footer**: Theme options
- **SocialMedia**: Displays social media icons with links
- **Tooltip**: A customizable tooltip component for additional information

## Tech Stack

- Svelte
- TypeScript
- Tailwind CSS (inferred from class names)

## Setup and Installation

1. Install the dependencies

```bash
npm install
```

2. Run project

```bash
npm run dev
```

3. Visit http://localhost:5173/
