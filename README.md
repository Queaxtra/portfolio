# Personal Portfolio Website

This is a modern, responsive portfolio website built with Svelte and TypeScript. It showcases my skills, provides contact information, and offers a customizable user experience.

## Features

- **Responsive Design**: Adapts seamlessly to different screen sizes
- **About Me Section**: Introduces the developer with a brief description and skills
- **Contact Information**: Easy access to various contact methods via a popup
- **Social Media Links**: Quick links to professional social media profiles
- **CV Download**: Direct link to download the CV

## Sections

1. **Home**: Introduces the developer with a brief description and call-to-action buttons
2. **About**: Provides more detailed information about the developer and showcases skills
3. **Contact**: A popup with multiple ways to get in touch

## Components

- **Skills Grid**: Displays a grid of skills with proficiency levels
- **Contact Popup**: A modal that appears when the contact button is clicked
- **Social Media Links**: Links to Telegram, LinkedIn, and X (formerly Twitter)

## Tech Stack

- Svelte
- TypeScript
- Tailwind CSS

## Key Features

- Responsive layout using Tailwind CSS classes
- Dynamic skills display using Svelte's each block
- Popup functionality with fade transition
- Scroll lock when popup is open

## Setup and Installation

1. Install the dependencies:

```bash
npm install
````

2. Run the project:

```bash
npm run dev
```

3. Visit `http://localhost:5173/` in your browser
